# radius-spring-boot-starter
radius server and client starter for spring-boot project

radius-spring-boot-starter is a pure java implimentation of radius server and client worked with springboot, which based on an open-souced project that I can barely remember the name, please contact me if anyone knows.

## To use this starter
### 1.Import maven cordinate in your project.
```
<groupId>org.tonythemesh</groupId>
<artifactId>radius-spring-boot-starter</artifactId>
<version>1.0-SNAPSHOT</version>
```
### 2.Implements AbstractRadiusServer and Dictionary.
### 3.Add @RadiusServer on your implementation of AbstractRadiusServer.
### 4.Do some configurations in your application.yml.
